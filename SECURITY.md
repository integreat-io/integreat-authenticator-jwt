# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.5.x   | :white_check_mark: |
| < 0.5   | :x:                |

## Reporting a Vulnerability

To report a vulnerability, please [create an issue](https://github.com/integreat-io/integreat-authenticator-jwt/issues) in this repo.
